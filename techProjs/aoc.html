THE ART OF COMPRESSION

March, 2019
UBC, Vancouver, BC, Canada


The Art of Compression was a school project (or assignment, I would say) that a colleague and I did during Spring of 2019 while I was in an Computer Science Algorithms and Data Structures course. The project's objective was to create a C++ application that can manipulate images' pixels and their RGB values to create stunning mosaic-like effect.

Through the project, I learned a lot more about using/writing classes and recursive functions. Since we had to make use of our understanding about Trees, recursion came in very handy. Moreover, I leanred about using pointers and managing memory more efficiently.
For this project, we coded in C++ using a provided pixels manipulation library called HSLAPixel.
Further details can be found on my Github repository: https://github.com/tunglam2101/the-art-of-compression
