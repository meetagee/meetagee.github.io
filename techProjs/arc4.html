TECHNICAL PROJECTS


ARC4 Decryption was a school project (or assignment, I would say) I did during the Summer of 2019 while I was in a Digital Systems Design course. During this time, I was learning about FPGAs and this project was an application on ARC4 (which is a popular symmetric stream cipher, and was widely used in encrypting web traffic, wireless data) Decryption circuit.

Through this project, I learned a lot about coding in SystemVerilog and cleared many confusions related to Hardware Description Languages (HDLs) and programming languages, where the modules in HDLs run concurrently compared to the traditional sequential approaches. I was also very amazed by the power of FPGAs and how they can be the future's widely used technology where projects can be prototyped easily and quickly using only code.
For the hardware component, I used a DE1-SoC development kit running on ARM Cortex-A9.
For software, I coded the circuit and testbenches in SystemVerilog under the Ready-enable (Handshaking) Protocol. The code was simulated and synthesized using Modelsim and Quartus Prime Lite 17.1
Further details can be found on my Github repository: https://github.com/tunglam2101/arc4-decryption
