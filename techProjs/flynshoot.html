TECHNICAL PROJECTS

FLY N' SHOOT

August, 2019
FPT Telecom, Ho Chi Minh City, Vietnam


Similar to the other embedded game Breaker that I made, Fly N' Shoot was also an embedded game that a colleague and I were working on during my internship period at FPT Telecom's IOT and Testing Department (Ho Chi Minh City, Vietnam). Unlike the other game, Fly N' Shoot' consists of many more active objects and thus, a bit more challenging but fits the purpose of learning Event-driven architecture really well. The game's objective is to destroy as many mines and keep the spaceship survived as long as possible.

Similarly, I learned a lot more about embedded systems through this project. However, since I was working with a colleague in this project and we were working on different parts/objects of the game, I got to learned a lot more about version control, Git branching and merging.
For the hardware component, we used a development kit running on TM32L (ARM Cortex-M3) MCU.
For software, we coded the game in C/C++ using a framework written by my instructor called Active Kernel. The game also runs in an Event-driven architecture.
Further details, including the schematics and related documents can be found on my Github repository: https://github.com/tunglam2101/flynshoot-embedded
