Similarly to the other embedded game Breaker that I made, Fly N' Shoot was also an embedded game that a colleague and I were working on during my internship period at FPT Telecom's IOT and Testing Department (Ho Chi Minh City, Vietnam). However, the game consists of many more active objects and thus, a bit more challenging but fits the purpose of learning Event-driven architecture really well. The game's objective is to destroy as many mines and keep the spaceship survive as long as possible.
For the hardware component, we used a development kit running on TM32L (ARM Cortex-M3) MCU.
For software, we coded the game in C/C++ using a framework written by my instructor called Active Kernel. The game also runs in an Event-driven architecture.
Further details, including the schematics and related documents can be found on my Github repository: https://github.com/tunglam2101/flynshoot-embedded
