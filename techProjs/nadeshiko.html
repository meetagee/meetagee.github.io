Nadeshiko was a personal project that I did during 2019 Winter break. It is basically a Discord bot built in Python using the discord.py library.

This was a pair programming project that I did with a close friend of mind. He as well wanted to learn more programming languages so hey, Python was a great playground here.
A bit background on the name Nadeshiko. I took the name from the character Nadeshiko Kagamihara (yes, I know what you're thinking - this guy is a weeb ( ͡° ͜ʖ ͡°). But she is caring and adorable - basically the characteristics of an ideal assistant for one. Anyway...) after finishing a very cozy anime series called Yuru Camp△.
The motive for me to make it is because I use Discord very often for chatting with my friends, not only while playing video games but studying as well. Sometimes we even have study groups via Discord and so I thought - why not making a bot that plays music and helps you doing Wolfram Alpha queries for your math homework?
We only had a 2-week winter break to work on it before a busy term began so it is faily simple. Though, the bot is able to find/query/play music from Youtube keywords/links. Moreover, it can do Wolfram Alpha query and make some meme-related materials just so you can make fun of your friends... I worked mainly on the music player. It was fairly tricky even though I found tons of tutorials/examples online. However, they were kinda outdated and so I had to dig in into it. Though, fun project it was!
Further details can be found on my Github repository: https://github.com/tunglam2101/Nadeshiko
