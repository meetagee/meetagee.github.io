NIXIE CLOCK

Ho Chi Minh City, Vietnam

With case - 1

With case - 2

Before proper casing

Nixie Clock was a personal project that I made during 2019 Winter break. I have always been a fan of it and wanted to build one myself using my knowledge with Arduino, C and C++.

I have always been a fan of nixie clocks, especially when I finished the 2 seasons of Steins;Gate, where it is called the Divergence Meter.
I started the project during summer of 2019. Unfortunately, one of the components (Nixie driver IC - SN74141) that I bought went horribly broken so the clock was not functioning correctly and it was about the time I fly back to Canada for my study. Luckily, with a bit of my dad's help in getting the new PCBs as well as the components, I was able to finish and brag about it <(￣︶￣)>.
Well, the nixie clock is fully built based on an open source project that I found online. It was built with 6 IN-12A bulbs (with some really cool RGB backlight) driven by HV5122 ICs. And to make it even better, the code was open source too! (づ ◕‿◕ )づ All I needed to do were soldering the components, loading the Arduino example, understanding what the code was doing and finally integrating the RTC so that it turns into a real clock.
Details can be found on Doayee's Github repository (once again, all the credits goes to this awesome guy for such an awesome open source project): https://github.com/doayee/Nixie
