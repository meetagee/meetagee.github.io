TECHNICAL PROJECTS


RC Car was a collab project that I did with a colleague. The idea was to make a RC car with minimum budget. The whole project was done using Arduino and a little bit of C.

We built the car using Arduino Uno (Atmega328p) and implemented the remote control functionalities via Wifi using a ESP3266 Wifi Shield. The main controller was rather simple as we only needed to tune the digital/analogWrite variables. However, the harder part was dealing with the ESP3266 wifi shield. We used Espressif Systems's AT Instruction Set to control and a TCP socket to send data.
The car can be controlled using WASD via terminal. Our plan was to create a mobile app for a more user-friendly experience but due to work and university courses, this will be implemented as soon as possible!
Further details on the implementation can be found in our open-source repository: https://github.com/tunglam2101/rc-car. All PRs/suggestions are welcomed :)!
