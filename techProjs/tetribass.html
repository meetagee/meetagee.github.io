TECHNICAL PROJECTS


TetriBASS is a group project that 2 other colleagues and I have been working on since summer of 2020. I am a fan of the game Tetris. I used to spend lots of hours on playing it on the site Tetris Friends. Though, starting May 2019, it was permanantly shut down and I wanted to work on a summer project so here it goes! Plus, Tetris Friends used Adobe Flash, which is technically discontinued so implementing the game on a different platform sounds extremely exciting to me. We implemented the game using React and TypeScript. The idea is to mimic the functionalities of Tetris Friends with some extensions on game modes, etc.

My main role in the project is to implement the basic game logic library, including the hard drop, ghost tile functionalities, etc. It was pretty challenging as it was the very first time I got to deal with React and TypeScript. Our initial plan was to simply use a pre-existing React Tetris library instead. However, since we would like to implement a lot more features to the game, it would be beneficial to create one our own.
We also tried to utilize and manage our project using Scrum (as I was in my coop term at the moment so I tried to apply as much as I learned from it!) and it turned out to be very effective.
Despite the main logic of the game is working, we still haven't got the time to dig deeper into the multiplayer part so it is still a very long way to go. However, it has been a very interesting project so far for me!
Further details on the implementation can be found in our open-source repository: https://github.com/NunkuGemu/TetriBASS. All PRs/suggestions are welcomed :)!
