Mobile Vault was a group design project that 5 more colleagues and I were working on during a Computer Engineering Design Studio course. Our mission was to integrate all that we had learned so far on Python, Raspberry Pi and sensors to create a small-scaled application that works. My roles in the project were developing the Frontened and verifying the integration of hardware (sensors, etc.) and software (Python).
Our idea was to make a portable safe (or a security system) that can be controlled via a webpage. The Mobile Vault's functionalities consist of real-time image capture of its' front door, alarm system and a passcode-controlled locking system.
For the hardware components, we used a Raspberry Pi 2 Model B along with various componenets such as camera, motion detector, piezo buzzer, LCD, keypad, etc.
For software, we coded in Python and JavaScript, HTML using Bootstrap 4, Ajax for Frontend Development and Django for Backend Development (in which we store our data in simple .txt and .jpg files).
Further details, including a Fritzing file of the design can be found on my Github repository: https://github.com/tunglam2101/CPEN291-2018W/tree/master/Project-2
